<h1>vue-admin 操作说明</h1>

### 安装 node v16 + 、pnpm v6 +

## 配置 node、pnpm

### 1，查看当前 npm 源

npm config ls

### 2，配置官方源

npm config set registry https://registry.npmjs.org

### 3，如果您还没安装 pnpm，请执行下面命令

npm install -g pnpm

## 用法

### 1, 安装依赖

pnpm i

### 2, 安装一个包

pnpm add 包名

### 3, 卸载一个包

pnpm rm 包名
